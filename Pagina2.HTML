Classes

Esclarescimento de Classes

A classe é a definição principal de o quê um personagem pode fazer

É mais do que uma profissão: é a vocação de um personagem. Elas costumam ser escolhidas de acordo com a raça optada por cada personagem.

Existem várias raças e cada uma delas possuem suas sinergias com as classes. Na tabela abaixo, podemos ver quais raças são mais recomendadas para cada classe.

MULTICLASSES

As classes podem ser combinadas entre si para criar personagens mais poderosos ou que cubram mais de uma função em uma aventura. É recomendado que as combinações de classe escolhidas acompanhem sempre os mesmos atributos para que seu personagem não fique desbalanceado, mas é apenas uma sugestão.

Abaixo podemos ver quais são as opções de multiclasse que mais funcionam no RPG. Lembre-se que a diversão vem em primeiro lugar, então toda combinação é válida mesmo que a sinergia não seja tão boa.

Opções de Milticlasses

  1. Feitiçeiro + Bruxo
  2. Feitiçeiro + Paladino
  3. Paladino + Bardo
  4. Clérigo e Druida
  5. Clérigo + Mago
